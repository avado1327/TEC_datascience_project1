Funciones para el clculo de la densidad del aire

Para calcular la densidad del aire utilizando la frmula del CIPM 2007 es necesario tener sumo cuidado con las unidades. Puesto que para calcular algunos parmetros las frmulas utilizan la temperatura en C o en K, la presin atmosfrica en hPa o en Pa y la humedad relativa como un valor entre 0 y 1 o como un valor entre 0 y 100.

Puede realizar los cambios que considere pertinentes

Psv=function(t){
  # Funcin que calcula la presin de vapor de saturacin siguiendo la frmula que se presenta en el artculo de referencia.
  #Input:
  #-------------
  #t: temperatura en C.
    
  #Output:
  #-------------
  #Psv: valor de la presin de vapor de saturacin en hPa.
  
  A = 1.2378847 * 10^(-5) #K^-2
  B = -1.9121316 * 10^(-2) #K^-1
  C = 33.93711047
  D = -6.3431645 * 10^(3) #K
  Tk = 273.15 + t #K
  Psv = 1 * exp(A*(Tk)^2 + B*Tk + C + D/Tk) 
  return(Psv/100)
}
f=function(t,P){
  # Funcin que calcula el factor de fugasidad siguiendo la ecuacin de la referencia.
    
  # Input:
  # --------------
  # P: presin en Pa.
  # t: temperatura en C.
  
  # Output:
  # --------------
  # f: Valor del coeficiente de fugacidad.

  alpha = 1.00062
  beta = 3.14 * 10^(-8) #Pa^(-1)
  gamma = 5.6 * 10^(-7) #K^(-2)
  
  f = alpha + beta*P + gamma*(t^2)

  return(f)
}
xv=function(t,P,h){
  # Funcin que calcula la fraccin molar de vapor de agua siguiendo el artculo de referencia.
    
  # Input:
  # ---------------
  # h: humedad relativa como un valor entre 0 y 1
  # P: presin atmosfrica en Pa.
  # t: Temperatura en C.
  
  # Output:
  # ---------------
  # xv: Valor de la fraccin molar de vapor de agua.
  f = f(t,P)
  Psv = Psv(t)
  xv = (h*f*Psv)/(P)
  return(xv)
}
Z=function(t,P,h){
  # Funcin que calcula el factor de compresibilidad con la ecuacin del artculo de referencia.
    
  # Input:
  # --------------
  # P: Presin atmosfrica en Pa.
  # h: Humedad relativa como un valor entre 0 y 1.
  # t: Temperatura en C.
  
  # Output:
  # ---------------
  # Z: factor de compresibilidad.

  a0 = 1.58123 * 10^(-6) #KPa^(-1)
  a1 = -2.9331 * 10^(-8) #Pa^(-1)
  a2 = 1.1043 * 10^(-10) #K^(-1)Pa^(-1) 
  b0 = 5.707 * 10^(-6) #KPa^(-1)
  b1 = -2.051 * 10^(-8) #Pa^(-1)
  c0 = 1.9898 * 10^(-4) #KPa^(-1)
  c1 = -2.376 * 10^(-6) #Pa^(-1)
  d = 1.83 * 10^(-11) #K^(2)Pa^(-2)
  e = -0.765 * 10^(-8) #K^(2)Pa^(-2)
  
  xv = xv(t,P,h)
  Tk = 273.15 + t
  Z = 1 - P*(a0 + a1*t + a2*(t^2) + (b0 + b1*t)*xv + (c0 + c1*t)*(xv^2))/(Tk) + P^2*(d + e*(xv^2))/(Tk^2)

  return(Z)
}
DensidadCIPM=function(t,P,h){
  # Funcin que calcula la densidad del aire a partir de la ecuacin del artculo de referencia.
    
  # Input:
  # --------------
  # P: Presin atmosfrica en Pa.
  # h: Humedad relativa como un valor entre 0 y 1.
  # t: Temperatura en C.
  
  # Output:
  # --------------
  # rho: densidad del aire en kg/m^3.

  Ma = 28.96546 * 10^(-3) #kg mol^(-1)
  Mv = 18.01528 * 10^(-3) #kg mol^(-1)
  R = 8.314462618 #J mol^(-1) K^(-1)
  Tk = 273.15 + t #K
  Z = Z(t,P,h)
  xv = xv(t,P,h)
  
  rho = (P*Ma)/(Z*R*Tk) * (1 - xv*(1 - (Mv/Ma)))
  return(rho)
}
# Calcule las densidades
DensidadDf=function(t,P_hpa,h_100){
  # Funcin que calcula la densidad del aire a partir de la ecuacin del artculo de referencia, considerando unidades usadas en el dataframe.
    
  # Input:
  # --------------
  # P_hpa: Presin atmosfrica en hPa.
  # h_100: Humedad relativa como un valor entre 0 y 100.
  # t: Temperatura en C.
  
  # Output:
  # --------------
  # rho: densidad del aire en g/cm^3.
  
  P = P_hpa * 100 #Presin en Pa
  h = h_100 / 100 #Humedad relativa como valor entre 0 y 1
  rho = DensidadCIPM(t,P,h) / 1000 #Densidad del aire en g/cm^3
  return(rho)
}
densidades=DensidadDf(df$Temperatura,df$Presin,df$Humedad)

De lo anterior se puede observar que: (1) La forma de los histogramas claramente no siguen la forma de la distribucin normal, tal y como se esperaba por los grficos Q-Q, (2) El perfil que siguen las columnas de los histogramas presenta, aparentemente, dos mximos locales en aproximadamente 0.001039 g/cm^3 (moda menor) y 0.001046 g/cm^3 para el caso nocturno y para el caso diurno en 0.001037 g/cm^3 (moda menor, mas sutil que en el caso nocturno) y 0.001047 g/cm^3 (moda mayor), (3) Ambos grupos tienen medias y mximos iguales y valores muy similares para sus medianas y cuartiles, (4) Sus distribuciones no son simtricas relativo a sus medias o medianas y tienden a tener una oblicuidad (skewness) negativa, (5) La mayora de los datos se distribuyen mayormente alrededor de las modas de las distribuciones con una kurtoris positiva.

La similitud de los datos (diurnos y nocturnos) nos lleva a preguntarnos si existen diferencias significativas entre ambos grupos. Segn lo que se explor anteriormente, se podra contestar negativamente respecto a la afirmacin sobre la existencia de dichas diferencias significativas, sin embargo, se realizar una prueba estadstica para confirmarlo. Como sus distribucin, sin lugar a duda, no es normal, se debe realizar una prueba estadstica no-paramtrica, en este caso se utilizar el test de Wilcoxon (como es para dos grupos, Test U de Mann-Whitney). Se utilizarn como hipotesis:

\(H_{0} =\) Existen diferencias significativas entre los conjuntos de datos \(H_{1} =\) NO existen diferencias significativas entre los conjuntos de datos

wilcox.test(df_noc2$Densidad,df_dia2$Densidad)
## 
##  Wilcoxon rank sum test with continuity correction
## 
## data:  df_noc2$Densidad and df_dia2$Densidad
## W = 1.1067e+11, p-value < 2.2e-16
## alternative hypothesis: true location shift is not equal to 0

Si se asume un nivel de significancia de \(\alpha = 0.05\) entonces se decide rechazar la hipotesis nula y se concluye que no hay diferencias significativas entre la densidad medida en los horarios diurnos y nocturnos.
